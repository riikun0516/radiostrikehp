どうもりーくんです。

  
  
  
  
  
  
  
  

本日は大学生になる人におすすめするパソコンのOSについて説明していく

  
  
  

まず大事なことを伝えておく。

  
  
  

それは「なにに使う」だ。

  
  
  

大学生でおすすめされるOSはざっとまとめるとこんな感じ

  
  
  

- Windows OS
- mac OS
- Chrome OS

  
  
  

の3つになるかと思うが、全てが同じことのできるわけではない。

  
  
  

ましてやそれぞれ値段がある。

  
  
  

値段で決めてたら後で痛い目に合うかもしれない。

  
  
  

なのでしっかりと説明しておこう。

  
  
  

## 症例1.いろんなことをする学部

  
  
  

この学部ではパソコンを何に使うかはわからない。そんな人ですかね・・。

  
  
  

その人におすすめしたいのは断然Windows OS。

  
  
  

Windowsは入れれるソフトウェアがたくさんあり、購入後でもこれがいるんだったって思ったとき大体Windows使っておけばどうにでもなります。

  
  
  

## 症例2.パソコンを全く使わない学部

  
  
  

パソコンを全く使わない学部ではChromeOSもしくはWindowsをおすすめします。

  
  
  

Chrome OSはLinuxをベースとした比較的軽めなOSです。かつ、開発元がGoogleであることからAndroidのアプリストアであるGoogle Play Storeが使える機種も少なくはありません。しかしSDなどでの論文提出になったときは困るかもしれません(中の人はChrome OSは使ったことないので)そうなっても困るので中の人の保険のために一応Windowsもおすすめします。

  
  
  

## 症例3.動画編集や写真編集などクリエイターに特化した学部

  
  
  

こちらはmac OSがおすすめです。というよりかなんでmac OSっていう表記にしてるかと疑問に思ってる人が多いと思いますので説明させていただきます。

  
  
  

mac OSが初めて人工知能siriに対応した、mac OS 10.12 SierraからAppleが何故かMac OS Xからmac OSに変えたから私もそちらの仕様変更に準拠させていただきそう表記させていただきます。

  
  
  

さてmac OSをおすすめするのは、動画編集ソフトなどは比較的機種性能に左右されてきます

  
  
  

mac OS搭載PCというのは、mac bookシリーズでしかない(Hackintoshをやってる強者を除く)のでOSの更新にソフトウェア側がいち早く対応できるので、性能を発揮しやすいからです。

  
  
  

さていかがだったでしょうか。
