#### 確認事項

  
  
  
  

  
2. SoftBankやY!Mobileですか?
  
4. 前使っていたのはキャリアもしくはSIMフリーiPhoneまたはキャリアiPhoneですか?
  

  
  
  
  

両方当てはまる場合は交換が必要になります。

  
  
  
  

SoftBankやY!Mobile(Y!Mobileの方はうろ覚えなのですが・・・。)端末によってSIMを分けています。

  
  
  
  

というわけでSIMの分け方を見ていきましょう

  
  
  
  

#### SIMカードの分け方(ソフトバンク編)

  
  
  
  

  
2. キャリア・SIMフリーiPhone
  
4. FeliCa対応Android
  
6. FeliCa未対応Android
  
8. 何でも使える万能SIM
  

  
  
  
  

こんな感じに説明したほうがわかりやすいのでやってますが、iPhoneSIMは何でも使えるみたいです。

  
  
  
  

#### 結局どうしたらいいの?

  
  
  
  

結局は何でも使える万能SIMに変えなくてはいけません。

  
  
  
  

変えるのは機種変更扱いとなりますので注意が必要です

  
  
  
  

料金はSIM交換手数料3,000円+契約事務手数料3,000円つまり6,000円かかります。

  
  
  
  

一部サービスも使えなくなります。

  
  
  
  

#### 使えなくなるサービス

  
  
  
  

  
2. おうち割光セット
  
4. Yahoo!JapanIDスマートログイン
  
6. Yahoo!プレミアムソフトバンク特約
