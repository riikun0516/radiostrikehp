# NTTドコモ、コーポレートサイト、MydocomoにおいてURLを変更

 株式会社NTTドコモはコーポレートサイト(旧URL:www.nttdocomo.co.jp)とMydocomo(旧URL:www.nttdocomo.co.jp/mydocomo)のURLを変更した。本日から変更されており。

dアカウントのログインにかかわるブラウザ側の仕様変更に対応した形となる。

新URLはコーポレートサイトが

[https://www.docomo.ne.jp](https://www.docomo.ne.jp/)

Mydocomoが

[https://www.docomo.ne.jp/mydocomo](https://www.docomo.ne.jp/mydocomo)

となる

しばらくの間は旧URLにアクセスしても新URLにリダイレクト(転送)される仕組みとなっている。

# JR西日本コーポレートサイト一部変更か

鉄道大手の西日本旅客鉄道株式会社はコーポレートサイトを一部変更したことが明らかとなった。

筆者がWayBackMachineの記録を確認したところ3月11日ごろに変更されているとみられており、サイト上部はJR西日本の様々な事業をわかりやすくまとめられた映像を流していたが、画像のスライドとなっている。

近年低容量のスマホプランが相次いで人気なため通信容量を使う映像ではなく画像のスライドショーを使うのがいいのではという判断になったのか、JRおでかけネットに要望が来たのかいずれかの理由により変更されたとみられている。
